    Inspector (Zones tab)

    With Nav = orbit, click a placement in the 3D view to inspect it: placement, shape, state record and behavior/hotspots.

    Entity browser (Zones tab)

    Toggle the sidebar list to Entities to list the zone's placements. The Show dropdown filters by kind (containers, text objects, hotspot tiles); the filter box matches shape name and #index. Click a row to inspect it and move the camera to it.

    ?list=entities opens the entity list; ?select=<i> auto-opens a placement; ?selectat=<x>,<y> picks the nearest; ?chapter=<n> sets the chapter; ?tab=<name> opens a specific inspector tab; ?enc=1 shows encounter markers.

    Dialog graph (Dialog tab)

    A conversation renders as a raw, flat tree: every node is a card (colored by class) and every edge a labeled arrow (if night, else, chapter == 1, keyword topics, and ⏫ chain page-turns). Nothing is collapsed or capped by default — a conversation that chains into the keyword web pulls the whole reachable subgraph.

    A switch (a branch that tests one state with mutually-exclusive arms) renders as an amber ⏃ state? card; its arm edges (= 1, 3..6, else) are hue-tinted so parallel storylines are traceable. Click an arm pill to collapse that arm to a dashed stub (⊕ = 2 · N nodes); click the stub to re-expand. Shared nodes reachable from other arms stay.

    A shared node (reached by two or more parents) creates a MERGE that makes the graph read as a DAG rather than a tree. Select it, then use the crumb bar’s merge: controls to break the merge: unmerge clones the node’s self-contained leaf block once per parent (dashed teal ⧈ clone cards) — allowed only when the block is entered solely through the shared node and never leaks out (otherwise an inline ⚠ error names the offending edge); reference shows the block once, detached, with a dashed ⠿ reference link from each parent (click a link to jump to the real block). Marks persist with the manual layout.
